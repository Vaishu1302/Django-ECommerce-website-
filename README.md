# 🛒 Django E-Commerce Website

A full-stack **E-Commerce Web Application** built using **Django**, **MySQL**, **Bootstrap 5**, and **Chart.js**. The application provides a complete online shopping experience with secure user authentication, product management, shopping cart, wishlist, order management, reviews, analytics dashboard, and admin features.
